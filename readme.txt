== Description ==
Join Us plugin gives an ability to create a gallery of users who are going to visit your event using Facebook login. Enable your guests to use also Linkedin, Google and Instagram with our PRO version.

If you create an event and want to create a gallery of users who go to the event, our plugin is suitable for this.

<h3>Features</h3>
<ul>
<li>Frontend gallery </li>
<li>Shortcode</li>
<li>Join us button</li>
<li>Facebook login</li>
<li>Collect list of users</li>
</ul>

== Installation ==

1.	Upload the "Joinus" folder to the /wp-content/plugins/ directory
2.	Activate the plugin through the 'Plugins' menu in WordPress
3.	Go to Dashboard > Settings > Join Us Plugin Settings
